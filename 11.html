# 🌟 Welcome to the ANOVA Universe! 🌟

*Where statistics meets creativity and data tells beautiful stories*

**Analysis of Variance**: The elegant dance between categorical predictors and continuous outcomes

## 🎯 What is ANOVA?

**ANOVA** (Analysis of Variance) is your statistical compass when navigating the relationship between:

- **🏷️ Independent Variable(s)**: Categorical (gender, treatment type, region)

- **📊 Dependent Variable**: Continuous (income, test scores, height)

*Think of it as asking: “Do different groups have meaningfully different averages?”*

# 🚀 Part I: One-Way ANOVA Adventure

## 🤔 The Big Question

**ANOVA tests**: Do the means of our dependent variable differ significantly across the levels (groups) of our categorical variable?

**🎓 Example**: Do average exam scores differ across 3 teaching methods?

- Teaching method → categorical (independent) ✨

- Exam score → continuous (dependent) 📈

## 📋 The Holy Trinity of Assumptions

### 🎯 The Three Pillars of ANOVA

1. **🔔 Normality**: Each sample comes from a normally distributed population

2. **🎲 Independence**: Each sample is drawn independently

3. **⚖️ Equal Variance**: Variance should be similar across groups

## 🧮 The Main Event: One-Way ANOVA

```{r one-way-anova}

# 🎭 The grand performance

one_way_anova <- aov(Sepal.Length ~ Species, data = iris)

summary(one_way_anova)

```

**🎉 Results Interpretation:**

P-value < 0.05 → We reject H₀!

**Conclusion**: There is a statistically significant difference between the mean sepal lengths across iris species at 5% significance level. Our botanical friends are indeed different! 🌺

## 🔍 Post-Hoc Detective Work

When ANOVA says “there’s a difference somewhere,” post-hoc tests tell us “here’s exactly where!”

### 🕵️ Pairwise T-Tests

```{r pairwise-tests}

# 🔍 Bonferroni correction - being extra careful

pairwise.t.test(iris$Sepal.Length, iris$Species, p.adj = “bonferroni”)

```

### 🎯 Tukey’s Honest Significant Differences

```{r tukey-hsd}

#| fig-cap: “Tukey HSD confidence intervals - visual proof of differences”

tukey_results <- TukeyHSD(one_way_anova)

tukey_results

# 🎨 Visualize the differences

plot(tukey_results,

col = “steelblue”,

main = “🎯 Tukey HSD Confidence Intervals”)

```

# 🔬 Part II: Assumption Validation Laboratory

## 🧪 Testing Normality

### Method 1: QQ Plot Artistry

```{r qq-plot}

#| fig-cap: “Q-Q Plot: Where theory meets reality in beautiful harmony”

qqPlot(one_way_anova$residuals,

id = FALSE,

col.lines = “red”,

col = “steelblue”,

pch = 19,

main = “🎨 Q-Q Plot: Normality Check”,

xlab = “Theoretical Quantiles”,

ylab = “Sample Quantiles”)

```

### Method 2: Shapiro-Wilk Test

```{r shapiro-test}

# 🧪 The formal test

shapiro_result <- shapiro.test(one_way_anova$residuals)

shapiro_result

```

**🎯 Normality Verdict**:

P-value > 0.05 → We don’t reject H₀. The residuals follow approximately normal distribution! ✅

### Method 3: Group-wise Normality Check

```{r group-normality}

# 🔬 Individual species examination

iris %>%

group_by(Species) %>%

summarize(

statistic = shapiro.test(Sepal.Length)$statistic,

p.value = shapiro.test(Sepal.Length)$p.value

) %>%

knitr::kable(

caption = “🌺 Normality Test by Species”,

digits = 4

)

```

## ⚖️ Testing Homogeneity of Variances

### 🔍 Bartlett’s Test

```{r bartlett}

bartlett.test(Sepal.Length ~ Species, data = iris)

```

### 🎯 Levene’s Test (More Robust)

```{r levene}

leveneTest(Sepal.Length ~ Species, data = iris)

```

## 🛟 When Assumptions Fail: Welch ANOVA

When equal variances assumption is violated, Welch ANOVA comes to the rescue!

```{r welch-anova}

library(rstatix)

# 🛡️ The robust alternative

waov <- iris %>%

welch_anova_test(Sepal.Length ~ Species)

waov

```

### 🎮 Games-Howell Post-Hoc

```{r games-howell}

# 🎯 Post-hoc for unequal variances

pwc_gh <- iris %>% games_howell_test(Sepal.Length ~ Species)

pwc_gh

```

# 🌟 Part III: Two-Way ANOVA Spectacular

**Two-Way ANOVA** examines:

1. 🎭 Main effect of Factor A

2. 🎪 Main effect of Factor B

3. 🎨 **Interaction effect** (A × B) - *The plot twist!*

## 💼 Job Satisfaction Case Study

```{r job-data}

library(datarium)

data(“jobsatisfaction”)

# 📊 Our new dataset exploration

jobsatisfaction %>%

group_by(gender, education_level) %>%

get_summary_stats(score, type = “mean_sd”) %>%

knitr::kable(

caption = “💼 Job Satisfaction by Gender and Education”,

digits = 2

)

```

### 📈 Sample Sizes Check

```{r sample-sizes}

table(jobsatisfaction$gender, jobsatisfaction$education_level) %>%

knitr::kable(caption = “🔢 Sample Sizes Across Groups”)

```

## 🎭 Two-Way ANOVA Performance

### 🎪 Main Effects Only

```{r two-way-main}

library(car)

score.lm <- lm(formula = score ~ gender + education_level, data = jobsatisfaction)

score.II.aov <- Anova(score.lm, type = 2)

score.II.aov

```

### 🎨 With Interaction Effect

```{r two-way-interaction}

score.lm.int <- lm(formula = score ~ gender * education_level, data = jobsatisfaction)

score.II.aov2 <- Anova(score.lm.int, type = 2)

score.II.aov2

```

## 🔍 Two-Way ANOVA Diagnostics

### 🧪 Normality Check

```{r two-way-qq}

#| fig-cap: “Q-Q Plot for Two-Way ANOVA residuals”

qqPlot(score.lm.int$residuals,

id = FALSE,

col.lines = “red”,

col = “steelblue”,

pch = 19,

main = “🎨 Two-Way ANOVA: Q-Q Plot”)

```

```{r two-way-shapiro}

shapiro.test(score.lm.int$residuals)

```

### ⚖️ Variance Homogeneity

```{r two-way-variances}

# 🔍 Bartlett test for interaction

bartlett.test(score ~ interaction(gender, education_level), data = jobsatisfaction)

# 🎯 Levene test

leveneTest(score ~ gender * education_level, data = jobsatisfaction)

```

## 🎭 The Interaction Drama

### 🤹 What Does Interaction Mean?

**Interaction is significant** = *The effect of one factor depends on the level of the other factor*

**Example**: Teaching method effectiveness might vary by gender:

- Strong effect for males, weak for females

- Or vice versa! 🎭

### 🔬 Simple Main Effects Analysis

```{r simple-main}

# 🎯 Group by gender, test education effect

model <- lm(score ~ gender * education_level, data = jobsatisfaction)

jobsatisfaction %>%

group_by(gender) %>%

anova_test(score ~ education_level, error = model)

```

### 🎪 Pairwise Comparisons

```{r pairwise-emmeans}

library(emmeans)

# 🔍 Detailed pairwise comparisons

pwc <- jobsatisfaction %>%

group_by(gender) %>%

emmeans_test(score ~ education_level, p.adjust.method = ‘bonferroni’)

pwc %>%

knitr::kable(

caption = “🎯 Pairwise Comparisons by Gender”,

digits = 3

)

```

**🎉 Key Finding**: Significant differences in job satisfaction scores between education levels for both males and females!

## 🌟 When Interaction is Not Significant

If no interaction exists, we focus on main effects:

```{r main-effects-focus}

# 🎯 Education level main effect

pairwise.t.test(jobsatisfaction$score,

jobsatisfaction$education_level,

p.adjust.method = ‘bonferroni’)

```

✅ **One-Way ANOVA**: Perfect for comparing multiple groups

✅ **Assumption Testing**: The foundation of reliable results

✅ **Post-Hoc Tests**: Finding exactly where differences lie

✅ **Two-Way ANOVA**: Exploring interactions and main effects

✅ **Robust Alternatives**: Solutions when assumptions fail

**Remember**: Statistics is not just about numbers—it’s about telling the story hidden in your data! 📊✨

## 📚 Quick Reference Guide

| Test | When to Use | Key Assumption |

|——|————-|—————-|

| One-Way ANOVA | 1 categorical IV, 1 continuous DV | Normality, Independence, Equal variances |

| Welch ANOVA | When equal variances assumption fails | Normality, Independence |

| Two-Way ANOVA | 2 categorical IVs, 1 continuous DV | Normality, Independence, Equal variances |

| Post-Hoc Tests | After significant ANOVA | Same as parent test |
